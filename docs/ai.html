AI 설정.

anf의 AI(요약·질문·이름 제안·자동 태그·정리·이미지 검색)는 기본 꺼짐입니다. 켜고 모델 하나만 연결하면 끝 — Claude가 가장 쉽고 강력합니다.

어떤 걸 고를까?

백엔드는 셋. 내용이 기기 밖으로 나가도 되는지와 가진 하드웨어로 고르세요.

제공자실행 위치필요한 것
Claude — 추천클라우드(Anthropic) · 사용량 과금API 키 + 결제
로컬 LLM내 Mac · 무료Ollama 또는 LM Studio 실행
Apple내 Mac · 무료macOS 26 + Apple Intelligence

"aiProvider": "auto"는 Claude(키가 있으면) → 로컬 서버(켜져 있으면) → Apple 순으로 시도합니다. "claude"·"local"·"apple"로 직접 고정할 수 있어요.

1. 설정 파일 열기

anf에서 ⌘, 또는 도구 메뉴 → AI 모델 연결…. 열리는 JSON 파일을 편집하고, anf으로 다시 전환하면 적용됩니다.

Claude (가장 추천)

1M 토큰 컨텍스트라 큰 문서·폴더도 통째로. 클라우드라 내용이 Anthropic으로 전송됩니다.

  1. console.anthropic.com → API Keys에서 Create Key. 키는 sk-ant-api03-… 형태입니다 (sk-ant-oat…는 OAuth 토큰이라 안 됩니다).
  2. 결제수단/크레딧이 등록돼 있어야 호출됩니다 (Billing).
  3. 설정 파일에 아래를 넣으세요:
"aiFeatures": true, "aiProvider": "auto", "aiApiKey": "sk-ant-api03-여기에-키"

모델을 비워두면 claude-opus-4-8(가장 깊은 추론)이 기본. "aiModel": "claude-sonnet-4-6"으로 바꾸면 더 빠르고 저렴합니다. Anthropic이 토큰 단위로 과금 — 요약 한 번은 1센트도 안 되고, 폴더 통째 질문은 더 듭니다.

로컬 LLM (Ollama · LM Studio)

기기 밖으로 안 나갑니다. Apple Intelligence 없이도 동작.

  1. 로컬 서버를 켜세요 — Ollama는 자동(:11434), LM Studio는 Developer → Start Server(:1234).
  2. 설정 파일:
"aiFeatures": true, "aiProvider": "local", "aiEndpoint": "http://localhost:11434/v1", "aiModel": "llama3.2"

LM Studio는 http://localhost:1234/v1. aiModel은 로드한 모델 이름 (LM Studio는 비워도 됨). 추천: llama3.2, qwen2.5, 또는 8B급 — 클수록 똑똑하지만 느리고, 비추론(plain) 모델이 가장 빠릿합니다.

Apple 온디바이스

번들·네트워크 0. macOS 26 + Apple Intelligence가 켜진 Mac이 필요합니다.

"aiFeatures": true, "aiProvider": "apple"

시스템 설정에서 Apple Intelligence를 켜고, Siri 언어가 맞아야 활성화됩니다.

설정 키 레퍼런스

AI 키는 전부 같은 설정 파일(⌘,)에 있습니다.

키값기본값
aiFeaturestrue / false — 마스터 스위치false
aiProviderauto · claude · local · appleauto
aiApiKeysk-ant-api03-… (Claude 전용)—
aiModel선택한 제공자의 모델 이름claude-opus-4-8
aiEndpointhttp://localhost:PORT/v1 (로컬 전용)—

클라우드(Claude)는 내용을 Anthropic으로 전송합니다. 로컬·Apple은 기기 밖으로 나가지 않습니다.

문제 해결

  • AI 메뉴가 비활성 → aiFeatures가 true가 아니거나, anf으로 다시 전환해 설정을 안 불러왔습니다.
  • Claude: 401 / "invalid x-api-key" → sk-ant-oat… OAuth 토큰을 넣었습니다. sk-ant-api03… API 키를 쓰세요.
  • Claude: "credit balance too low" → Billing에서 결제수단/크레딧을 추가하세요.
  • 로컬: "connection refused" → 서버가 안 켜졌거나 포트가 틀립니다 (Ollama 11434, LM Studio 1234). aiEndpoint 확인.
  • 로컬: "model not found" → aiModel이 받아둔/로드한 모델과 일치해야 합니다 (ollama list).
  • Apple: 사용 불가 → macOS 26 + Apple Intelligence 켜짐이 필요하고, Siri 언어가 내용과 맞아야 합니다.
  • 응답이 느림 → 로컬 추론 모델은 생각이 많습니다. 더 작은 비추론 모델을 쓰거나 Claude로.

그래도 막히면 GitHub Issues로 알려주세요.
