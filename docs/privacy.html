개인정보처리방침

시행일: 2026년 6월 12일

요약: anf는 개인정보를 수집하지 않습니다. 계정도, 텔레메트리도, 분석 도구도 없습니다. 모든 데이터는 사용자의 Mac에만 저장됩니다.

1. 수집하는 정보

없습니다. anf는 사용자의 개인정보를 수집·전송·판매하지 않으며, 계정 가입이 없고, 텔레메트리·사용 통계·크래시 리포트·광고 식별자를 일절 수집하지 않습니다.

2. 네트워크 통신

anf가 자발적으로 수행하는 네트워크 요청은 단 하나입니다:

  • 업데이트 확인 — 하루 1회 GitHub Releases API (api.github.com)에 최신 버전을 조회합니다. 이 요청에는 어떤 사용자 데이터도 포함되지 않으며, 일반적인 웹 요청과 마찬가지로 GitHub가 IP 주소를 볼 수 있습니다(GitHub 개인정보 정책 적용). 오프라인이면 조용히 건너뜁니다.
  • SSH / SFTP — 사용자가 직접 등록·연결한 호스트에만, 사용자가 연결할 때만 통신합니다. 인증은 키/에이전트 방식이며 anf는 비밀번호를 입력받지도, 저장하지도 않습니다.

3. 기기에 저장되는 데이터

아래 데이터는 모두 사용자의 Mac에만 저장되며 외부로 전송되지 않습니다:

  • 검색 인덱스(파일 경로 목록) — ~/Library/Caches/anf/
  • 설정(레이아웃, 핀, Workspace, 최근 폴더 등) — macOS UserDefaults
  • SSH 연결 다중화 소켓 — ~/.anf/
  • 진단 로그 — 기본 비활성. ANF_TRACE=1로 직접 실행한 경우에만 /tmp/anf-trace.log에 기록됩니다.

앱을 삭제하면 위 데이터를 함께 정리할 수 있습니다 (brew uninstall --zap --cask anf).

4. 제3자 제공

수집하는 정보가 없으므로 제3자에게 제공하는 정보도 없습니다.

5. 방침 변경

방침이 바뀌면 이 페이지와 GitHub 저장소에 변경 내역을 공개합니다.
