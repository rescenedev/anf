Privacy Policy

Effective: June 12, 2026 (web-analytics notice added June 14, 2026)

In short: The anf app collects nothing — no accounts, no telemetry, no analytics. All of your data stays on your Mac. (This website uses cookieless, anonymous visit stats only — see §5.)

1. What we collect

Nothing. anf does not collect, transmit or sell your personal data. There is no account to create, and it gathers no telemetry, usage statistics, crash reports or advertising identifiers whatsoever.

2. Network activity

anf makes exactly one network request on its own:

  • Update check — once a day it queries the GitHub Releases API (api.github.com) for the latest version. The request carries no user data; as with any web request, GitHub can see your IP address (their privacy policy applies). Offline, it is silently skipped.
  • SSH / SFTP — only to hosts you add and connect to yourself, only when you connect. Authentication is key/agent based; anf never prompts for or stores passwords.

3. Data stored on your device

The following stays on your Mac and is never sent anywhere:

  • Search index (a list of file paths) — ~/Library/Caches/anf/
  • Settings (layout, pins, Workspaces, recent folders, …) — macOS UserDefaults
  • SSH connection-multiplexing sockets — ~/.anf/
  • Diagnostic log — off by default. Only when you launch with ANF_TRACE=1 is anything written to /tmp/anf-trace.log.

Uninstalling can remove all of the above (brew uninstall --zap --cask anf).

4. Third parties

Since nothing is collected, nothing is shared with third parties.

5. About this website (separate from the app)

Sections 1–4 above are about the anf app, which contains no analytics of any kind. This promotional website (rescenedev.github.io/anf) uses Cloudflare Web Analytics for visit statistics:

  • It uses no cookies — nothing is stored on your device, and it does not identify you or track you across sites.
  • Only anonymous aggregates are recorded — page views, approximate location (country), referrer and browser type.
  • It is not Google Analytics, and no data is sold or shared for advertising or profiling.

This is unrelated to downloading or using the app; if you never visit the website, none of these stats occur.

6. Changes

If this policy changes, we publish the change here and in the GitHub repository.
